# Altboard

An Altoids tin sized circuit board for looking at my Pronote schedule!

## Hardware Info

### Buttons:

- GPIO 4: SELECT A
- GPIO 5: SELECT B
- GPIO 6: DOWN
- GPIO 7: RIGHT
- GPIO 15: LEFT
- GPIO 16: UP

### Display:

Data Bus

- GPIO 35: DB0
- GPIO 36: DB1
- GPIO 37: DB2
- GPIO 38: DB3
- GPIO 39: DB4
- GPIO 40: DB5
- GPIO 41: DB6
- GPIO 42: DB7
- GPIO 2: DB8

Interface Select

- GPIO 45: IM0
- GPIO 48: IM1
- GPIO 47: IM2
- GPIO 21: IM3

Other

- GPIO 14: RD
- GPIO 13: D/CX
- GPIO 12: WR
- GPIO 11: CS
- GPIO 10: FMARK
- GPIO 3: LCD_BKLT_PWM

Driver: ILI9341
